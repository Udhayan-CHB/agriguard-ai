# AgriGuard AI

Multi-Agent System for Sustainable Smallholder Farming.  
Addresses SDG 2, 12, 13, 15.  
Built with IBM Granite, LangGraph, FastAPI, Next.js, ChromaDB, and Watson Discovery.

## Quick Start

### Backend

d backend
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload

### Frontend

cd frontend
npm install
npm run dev

See [AGENTS.md](AGENTS.md) for full architecture and development guide.
